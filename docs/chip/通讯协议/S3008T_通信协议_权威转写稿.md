# S3008T 体脂仪 BIA 通信协议（PDF 权威转写稿）

> **来源**: `芯片模组/体脂模组/通信协议.pdf`（12 页扫描件，Page 2 含完整协议图）
> **转写日期**: 2026-07-11
> **芯片**: S3008T（东莞市智睿科技 SOP-8 生物阻抗测量 IC）
> **接口**: UART 3.3V TTL，兼容 RS232
> **波特率**: 9600 bps 默认，8N1
> **校验**: CRC-8（多项式 0x07，初值 0x00）

---

## 1. 协议总览

四电极人体生物阻抗测量芯片，基于 BIA（生物电阻抗分析）原理。
通过测量人体阻抗并结合身高/体重/年龄/性别等数据，间接推算体成分。

**注意**：本芯片**仅输出阻抗值**，19 项体成分需配合 BestHealth TwoLegs 算法 SDK 计算。
（参见 `BestHealth_TwoLegs_技术参考手册.md` §5.2 推荐部署架构）

---

## 2. 帧结构

```
发送帧(主机 -> 芯片): [Length] [CMD] [DataN] [CRC8]
返回帧(芯片 -> 主机): [Length] [CMD] [DataN] [CRC8]

- Length: 数据字节数（CMD + DataN + CRC8 的字节数）+ 1
- CMD:    命令号（1 字节）
- DataN:  N 字节数据
- CRC8:   CRC-8 校验（多项式 0x07，初值 0x00）
```

**关键点**：无独立帧头字节，**Length 即为帧首字节**，接收方以 Length 切帧。

---

## 3. 命令清单

| 命令号 | 名称 | 说明 |
|--------|------|------|
| 0x01   | 设置测量模式 | 测量/待机切换（Data0：0=待机，1=测量）|
| 0x02   | 设置测量频率 | Data0=频率索引（0=50KHz）|
| 0x03   | 设置体重 | Data0..1=体重×10（大端 16 位，如 60.5kg→605）|
| 0x04   | 设置身高 | Data0=身高（cm，90~220）|
| 0x05   | 设置年龄 | Data0=年龄（6~99）|
| 0x06   | 设置性别 | Data0=性别（0=女，1=男）|
| 0x07   | 设置用户类型 | Data0=用户类型（0=普通，1=运动员）|
| 0x08   | 读取测量结果 | 芯片返回 50KHz 阻抗值（Ω，大端 16 位）|

---

## 4. 典型时序（获取一次体成分）

```
1. 主机 -> 芯片:  [Length] 0x01 [0x01]   CRC8    # 进入测量
2. 主机 -> 芯片:  [Length] 0x02 [0x00]   CRC8    # 选 50KHz
3. 主机 -> 芯片:  [Length] 0x03 [DH DL]  CRC8    # 设体重
4. 主机 -> 芯片:  [Length] 0x04 [Height] CRC8    # 设身高
5. 主机 -> 芯片:  [Length] 0x05 [Age]    CRC8    # 设年龄
6. 主机 -> 芯片:  [Length] 0x06 [Gender] CRC8    # 设性别
7. 主机 -> 芯片:  [Length] 0x07 [Type]   CRC8    # 设用户类型
8. 主机 -> 芯片:  [Length] 0x08 []       CRC8    # 读取阻抗
9. 芯片 -> 主机:  [Length] 0x08 [DH DL]  CRC8    # 返回 50KHz 阻抗值
10. 主机调用 BestHealth TwoLegs SDK:  compute(height, weight, z_50K, age, sex, type) -> 19 项体成分
```

---

## 5. CRC-8 验算

**多项式**: `x^8 + x^2 + x + 1` (0x07)
**初值**: 0x00
**输入**: Length, CMD, DataN（除 CRC8 字节外所有字节）
**输出**: 1 字节校验

## 6. 与原项目代码差异

| 维度 | 原代码 (src/hardware/bia_*) | 本协议（权威） |
|------|------|------|
| Header | 0x55 / 0xAA 双向 | ❌ 无帧头 |
| Length | 整帧总长（含帧头）| CMD+DataN+CRC8 字节数 |
| 校验 | Sum-8（算术和）| **CRC-8 (poly=0x07)** |
| 波特率 | 115200 | **9600** |
| 命令 | 0xF0/0xB0/0xB1/0xC0-C3（BestHealth 假设）| **0x01-0x08（S3008T 真实）** |
| 体成分 | 0xC0 一帧全出 | **仅返回阻抗，19 项算法 SDK 算** |

**结论**: 原 BIA 模块代码基于 BestHealth 协议假设，**与 S3008T 真实协议不兼容**。
Step 2 已实现 MockTransport，因此 mock 测试通过；接入真实硬件前必须按本规范重写。
