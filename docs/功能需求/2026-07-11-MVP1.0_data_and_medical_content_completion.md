# MVP1.0 医学数据与指标补齐实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**目标:** 补齐 MVP1.0 全部 P0 医学参考数据（HRV/体脂率/肌肉量/水分率等 8 项指标校准），新增 3 条建议模板，建立专家评审框架，填充证据文献引用。

**架构:** 采用"数据校准 → 模板补齐 → 评审框架 → 验证测试"四阶段顺序推进。所有变更以 `data/knowledge/medical_knowledge.json` 和 `data/knowledge/advice_templates.json` 为核心交付物，评审框架以 `data/knowledge/reviews/` 目录结构为载体。

**技术栈:** Python 3.8+（项目运行时），JSON Schema v1.1（知识库格式），pytest 7.4+（测试框架）。

### Task 9: 生成中国人群 BIA 校准 CSV

**文件:**
- Create: `scripts/data/generate_bia_calibration_csv.py`
- Create: `data/knowledge/calibration/bia_calibration_china.csv`

**背景:** 基于 GASC 2025 [PMID:40620559] 数据生成中国人群 BIA 校准参考 CSV。MVP 阶段使用 BIA-on-BIA 参考（非 DEXA 校准），但标注偏差范围。v2.0 需补充 DEXA 配对数据。

- [ ] **Step 1: 创建生成脚本**

```python
"""scripts/data/generate_bia_calibration_csv.py

从 medical_knowledge.json 中的 chinese_reference 字段导出 BIA 校准 CSV。

输出: data/knowledge/calibration/bia_calibration_china.csv

列:
    gender,age_group,p50,p75_obesity,p90,bias_vs_dexa_note
"""

import csv
import json
import sys
from pathlib import Path

SRC = Path("data/knowledge/medical_knowledge.json")
DST = Path("data/knowledge/calibration/bia_calibration_china.csv")

def main():
    with open(SRC, encoding="utf-8") as f:
        kb = json.load(f)
    bf = kb.get("indicators", {}).get("body_fat_pct", {})
    cref = bf.get("chinese_reference", {})
    percentiles = cref.get("percentiles", {})
    bias_note = cref.get("bias_note", "")

    if not percentiles:
        print("❌ medical_knowledge.json 中缺 chinese_reference.percentiles")
        sys.exit(1)

    DST.parent.mkdir(parents=True, exist_ok=True)
    with open(DST, "w", newline="", encoding="utf-8") as csvfile:
        writer = csv.writer(csvfile)
        writer.writerow(["gender", "age_group", "bfp_p50", "bfp_p75_obesity", "bfp_p90", "bias_note"])
        for gender_key, age_groups in percentiles.items():
            gender_label = "male" if gender_key == "male" else "female"
            for age_key, vals in age_groups.items():
                writer.writerow([
                    gender_label,
                    age_key,
                    vals.get("P50", ""),
                    vals.get("P75_obesity", ""),
                    vals.get("P90", ""),
                    bias_note,
                ])
    print(f"✅ 已写入 {DST} ({sum(1 for _ in open(DST)) - 1} 行数据)")

if __name__ == "__main__":
    main()
```

def test_template_variables_match(templates):
    template_vars = set(templates.get("template_variables", {}).keys())
    used_vars = set()
    for rule in templates["rules"].values():
        template = rule.get("template", "")
        # 用正则提取 {var_name}
        for match in re.finditer(r"\{(\w+)\}", template):
            used_vars.add(match.group(1))
    # 忽略内置变量
    ignored = {"age", "gender_str"}
    undefined = used_vars - template_vars - ignored
    assert not undefined, f"Undefined template variables: {undefined}"
    # 检查未使用的 template_variables
    unused = template_vars - used_vars - {"reference_low", "reference_high", "reference_median"}
    if unused:
        print(f"⚠️ 未使用的 template_variables: {unused}")
